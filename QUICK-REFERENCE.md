# HopeVoice Hub - Quick Reference Guide

## 🚀 Getting Started
1. Visit the website
2. Accept privacy notice
3. Start using - no account needed!

## 🎯 Top Features at a Glance

### 🎙️ **VoiceUp** - Share Your Voice
- Record up to 2 minutes
- Add tags (help needed, urgent, etc.)
- Post anonymously if desired
- Browse community stories

### 📋 **Needs Board** - Get Help
- Post what you need (food, clothing, shelter, etc.)
- Set urgency level
- Helpers respond with "I Can Help"
- Track status (open → in progress → addressed)

### 🗺️ **Resource Radar** - Find Services
- Search for shelters, food banks, clinics
- Filter by category and status
- Get directions
- See hours and contact info

### ⚖️ **RightsBot** - Know Your Rights
- Ask questions about rights
- Get instant OHRC guidance
- Learn about sleep, shelter, discrimination laws
- Link to Incident Logger

### 📝 **Incident Logger** - Document Violations
- Record rights violations
- Include date, time, location, details
- Track status
- Export for legal use

### 👨‍⚖️ **Ask a Lawyer** - Legal Help
- Submit anonymous legal questions
- Get responses from verified lawyers
- Topics: housing, discrimination, rights

## 🔒 Privacy Features

### You Control Your Data
- ✅ Everything stored on YOUR device
- ✅ No external servers
- ✅ Post anonymously anytime
- ✅ Export your data (Admin → Export)
- ✅ Delete all data (Admin → Clear)

### Tips for Privacy
- Use anonymous posting for sensitive topics
- Export important records regularly
- Use incognito mode if on shared device

## ⚡ Quick Actions

### Record a Voice Story
1. Click VoiceUp
2. Click Start Recording
3. Speak (up to 2 min)
4. Click Stop
5. Add title & tags
6. Post

### Post a Need
1. Click Needs Board
2. Fill form (what, category, urgency)
3. Add location (optional)
4. Click Post Need

### Find a Resource
1. Click Resource Radar
2. Use search or filter
3. View details
4. Get directions

### Log an Incident
1. Click Incident Logger
2. Select type
3. Describe what happened
4. Add location
5. Submit

### Track Your Health
1. Click HealthLog
2. Enter sleep, stress, feeling
3. Add notes
4. Save Entry

## 🆘 Troubleshooting

### Can't Record Audio?

- Try different browser
- Close other apps using mic

### Data Not Saving?
- Check localStorage enabled
- Not in incognito mode?
- Try different browser

### Feature Not Working?
- Refresh the page
- Clear browser cache
- Update browser to latest
- Check internet connection

## 💬 Common Questions

**Q: Do I need to create an account?**  
A: No! Use immediately without signup.

**Q: Is my data private?**  
A: Yes! Everything stays on your device only.

**Q: Can I post anonymously?**  
A: Yes! Most features have anonymous option.

**Q: Will my data sync between devices?**  
A: Not currently. Export and import to transfer.

**Q: How do I backup my data?**  
A: Admin → Export All Data → Save file

**Q: Can others see my health log?**  
A: No! HealthLog is completely private.

**Q: What if I need urgent help?**  
A: Post on Needs Board with "critical" urgency. For emergencies, call 911.
